# DREAM O'SPACESHEEP
En este Bullet Hell de acción frenética, controlarás a Arthur, un informático estresado 
por lo exámenes que está teniendo pesadillas con sus temores de Fundamentos de programación.
Tendrás que subierte a lomos de Holly Molly, tú cordero de peluche, para derrotar tus peores males.

# Controles
## Teclado
- Movimiento: WASD / Flechas  
- Disparar:  Barra espaciadora
- Pausa: Esc 
## Mando
- Movimiento: Joystick izquierdo / Cruceta
- Disparar:  A/B/Y/X/RT/LT/RB/LB
- Pausa: Start/Select

## CHETOS
En el menú de opciones dentro de lnivel, puedes encontrar una variedad de modificaciones
que facilitan ver el todo el contenido sin apenas dificultad.
- Invencibilidad
- Munición infinita
- Animaciones secretas(muy recomendable probarlo al menos una vez)
